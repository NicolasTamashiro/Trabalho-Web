Perguntas Frequentes

Para quando é a entrega do projeto?

A entrega do projeto é para o dia 10/04/2026 até as 23:59 no horário de Brasília.

Como é para ser feito a entrega?

A entrega deve ser feita através da plataforma MOODLE, anexando o link do repositório do GITHUB.

O projeto pode ser feito indivualmente ou em grupo de até 2 pessoas no máximo.

Deverá ser apresentado no dia 16/04/2026 as 19:00 no horário de Brasília.
